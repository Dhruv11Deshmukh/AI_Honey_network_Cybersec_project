# AI_Honey_network_Cybersec_project
The Autonomous AI Honey-Network with Dynamic Deception is an advanced cybersecurity system designed to detect, analyze, and respond to cyberattacks using intelligent deception techniques. Traditional honeypots are static and can be easily identified by skilled attackers, making them less effective in modern security environments. This project enhances the concept by creating a dynamic, adaptive honeypot network powered by artificial intelligence.

The system deploys multiple simulated services, such as a fake SSH server and a mock IoT device, which act as decoys to attract potential attackers. When an attacker interacts with these systems, their actions are recorded and monitored in real time. The core innovation of this project lies in its AI-driven deception engine, developed using machine learning techniques. This engine generates realistic fake data, including credentials, files, and system responses, to mimic genuine system behavior. This keeps attackers engaged for longer periods, allowing deeper analysis of their strategies and tools.

In parallel, network traffic is captured and analyzed using tools like Wireshark through its command-line interface, TShark. The system identifies suspicious patterns such as repeated login attempts or unauthorized access. Based on this analysis, it automatically updates the system’s defense by adding firewall rules to block malicious IP addresses, ensuring real-time protection.

The project is implemented in Python, utilizing libraries such as PyTorch for AI modeling and Flask for simulating IoT services. It is executed in a Windows environment using a virtual environment for efficient dependency management. Overall, this project demonstrates a proactive cybersecurity approach by integrating deception, automation, and AI to enhance threat detection, analysis, and response capabilities effectively.
